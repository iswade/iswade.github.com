    拓扑排序

    1. 首先构造邻接表方式存储树
    2. 然后计算所有节点的入度，将入度为 0 的节点入队
    3. 循环从队列中取出元素，如果队列为空则退出
    4. 如果队列不为空，则遍历取出元素的邻接链表，如果入度为 0，将所有邻接元素的入度--
    5. 如果入度为 0，则将这个元素加入队列
    6. 重复3

    环形子数组的最大和

    注意是子数组，

    子数组的最大和 dp[i] = max(A[i], dp[i-1] + A[i])

    子数组的最小和 dp[i] = min(A[i], dp[i-1] + A[i])

    最终结果：total - min

    下一个排列算法

    1. 从后往前找到第一个升序元素（所在角标 k）
    2. 如果没有找到，则说明已经是有序的，比如{1,2,3}这种

    5. 将nums[k:]反向

    然后用 prev 删除第 n 个元素即可。
